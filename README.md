# GlobalStandard - Corporate Website

## Overview

本リポジトリは、ポートフォリオサイトを想定したWebサイトのコーディング実装です。  
デザインカンプをもとに、制作会社案件を意識した実務レベルでの実装を行っています。

以下の点を重視して構築しています。

- デザインの再現性（ピクセル精度）
- レスポンシブ対応（PC / Tablet / SP）
- JavaScriptによるインタラクション実装
- WordPress化を前提とした構造設計
- 保守性・拡張性を意識した設計
